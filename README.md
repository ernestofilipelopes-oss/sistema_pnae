# Sistema PNAE

## Overview
Sistema PNAE is a project designed to manage and facilitate the National School Feeding Program (PNAE) in Brazil. The application aims to streamline processes associated with food supply, tracking, and management for schools involved in the program.

## Features
- **User Management**: Allows administrators to manage users and their roles within the system.
- **Food Management**: Keep track of food supplies, nutritional information, and ordering processes.
- **School Management**: Manage information related to schools participating in the PNAE.
- **Report Generation**: Generate reports for food consumption, supply status, and budget analysis.

## Installation
To run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/ernestofilipelopes-oss/sistema_pnae.git
   ```
2. Navigate into the project directory:
   ```bash
   cd sistema_pnae
   ```
3. Install required dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
After starting the application, navigate to `http://localhost:3000` in your web browser to access the system.
